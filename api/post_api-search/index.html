/api/search

Search for the content

Params

Required:

  • type : "article" | "tutorial" (*)
  • query : string*

Example:

{
  "type": "article",
  "query": "с"
}

Response

  • With the correct set of arguments, it will return the type of search and list of articles
  • Return 400 if there's missing param in body
